# Créditos e Atribuições - Frontend

Este projeto utiliza software de código aberto desenvolvido pela comunidade. Agradecemos a todos os mantenedores e contribuidores das seguintes bibliotecas:

## Dependências Principais

### React e Ecossistema
- **React** (MIT) - Facebook, Inc. e colaboradores
  - Uma biblioteca JavaScript para construir interfaces de usuário
  - https://reactjs.org/

- **React Router** (MIT) - Remix Software, Inc.
  - Roteamento declarativo para React
  - https://reactrouter.com/

- **React Query / TanStack Query** (MIT) - Tanner Linsley e colaboradores
  - Gerenciamento poderoso de estado do servidor para React
  - https://tanstack.com/query

### UI e Estilos
- **Tailwind CSS** (MIT) - Tailwind Labs, Inc.
  - Framework CSS de utilitários
  - https://tailwindcss.com/

- **Headless UI** (MIT) - Tailwind Labs, Inc.
  - Componentes UI acessíveis sem estilos
  - https://headlessui.com/

### Formulários e Validação
- **React Hook Form** (MIT) - Bill Luo e colaboradores
  - Formulários performáticos e flexíveis com validação fácil
  - https://react-hook-form.com/

- **Zod** (MIT) - Colin McDonnell
  - Validação de esquemas TypeScript-first
  - https://zod.dev/

### Utilidades
- **Axios** (MIT) - Matt Zabriskie e colaboradores
  - Cliente HTTP baseado em promessas
  - https://axios-http.com/

- **date-fns** (MIT) - Sasha Koss e colaboradores
  - Biblioteca moderna de utilitários para datas em JavaScript
  - https://date-fns.org/

### Segurança
- **DOMPurify** (MPL-2.0 OR Apache-2.0) - Cure53 e colaboradores
  - Sanitizador de HTML e prevenção XSS
  - https://github.com/cure53/DOMPurify
  - Usado sob licença Apache-2.0

### Ferramentas de Desenvolvimento
- **Vite** (MIT) - Evan You e colaboradores
  - Ferramenta de construção rápida para desenvolvimento frontend
  - https://vitejs.dev/

- **TypeScript** (Apache-2.0) - Microsoft Corporation
  - Superset tipado de JavaScript
  - https://www.typescriptlang.org/

- **ESLint** (MIT) - Nicholas C. Zakas e colaboradores
  - Ferramenta de linting para JavaScript/TypeScript
  - https://eslint.org/

## GraphQL e Cliente Apollo
- **Apollo Client** (MIT) - Apollo Graph, Inc.
  - Cliente GraphQL completo para React
  - https://www.apollographql.com/docs/react/

## Ícones e Recursos
- **Heroicons** (MIT) - Tailwind Labs, Inc.
  - Belos ícones SVG feitos à mão
  - https://heroicons.com/
