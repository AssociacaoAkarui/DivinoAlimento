# Créditos e Atribuições - Backend

Este projeto utiliza software de código aberto desenvolvido pela comunidade. Agradecemos a todos os mantenedores e contribuidores das seguintes bibliotecas:

## Dependências Principais

### Framework e Core
- **Express** (MIT) - TJ Holowaychuk, Strong Loop, e colaboradores
  - Framework web rápido, minimalista e flexível para Node.js
  - https://expressjs.com/

- **Node.js** - OpenJS Foundation
  - Runtime de JavaScript construído sobre o motor V8 do Chrome
  - https://nodejs.org/

### GraphQL
- **Apollo Server** (MIT) - Apollo Graph, Inc.
  - Servidor GraphQL de produção para Node.js
  - https://www.apollographql.com/docs/apollo-server/

- **GraphQL** (MIT) - GraphQL Foundation
  - Linguagem de consulta para APIs
  - https://graphql.org/

### Banco de Dados
- **PostgreSQL Client (pg)** (MIT) - Brian Carlson e colaboradores
  - Cliente PostgreSQL para Node.js sem bloqueio
  - https://node-postgres.com/

- **Knex.js** (MIT) - Tim Griesser e colaboradores
  - Construtor de consultas SQL para Node.js
  - https://knexjs.org/

### Autenticação e Segurança
- **bcrypt** (MIT) - Nick Campbell e colaboradores
  - Biblioteca para hash de senhas
  - https://github.com/kelektiv/node.bcrypt.js

- **jsonwebtoken** (MIT) - Auth0, Inc.
  - Implementação de JSON Web Tokens
  - https://github.com/auth0/node-jsonwebtoken

- **cors** (MIT) - Troy Goode e colaboradores
  - Middleware CORS para Express
  - https://github.com/expressjs/cors

- **helmet** (MIT) - Adam Baldwin, Evan Hahn e colaboradores
  - Ajuda a proteger aplicações Express configurando headers HTTP
  - https://helmetjs.github.io/

### Validação e Parsing
- **body-parser** (MIT) - Colaboradores Express.js
  - Middleware de parsing de corpo de requisições
  - https://github.com/expressjs/body-parser

### Utilidades
- **dotenv** (BSD-2-Clause) - Scott Motte e colaboradores
  - Carrega variáveis de ambiente a partir do arquivo .env
  - https://github.com/motdotla/dotenv

- **lodash** (MIT) - John-David Dalton e colaboradores
  - Biblioteca de utilitários JavaScript moderna
  - https://lodash.com/

- **date-fns** (MIT) - Sasha Koss e colaboradores
  - Biblioteca moderna de utilitários para datas
  - https://date-fns.org/

### Logging e Monitoramento
- **winston** (MIT) - Charlie Robbins e colaboradores
  - Logger para quase tudo
  - https://github.com/winstonjs/winston

- **morgan** (MIT) - Colaboradores Express.js
  - Middleware de logging de requisições HTTP
  - https://github.com/expressjs/morgan

### Testing (DevDependencies)
- **Jest** (MIT) - Facebook, Inc.
  - Framework de testes delicioso para JavaScript
  - https://jestjs.io/

- **Supertest** (MIT) - TJ Holowaychuk e colaboradores
  - Biblioteca de testes HTTP de alto nível
  - https://github.com/visionmedia/supertest

### Ferramentas de Desenvolvimento
- **TypeScript** (Apache-2.0) - Microsoft Corporation
  - Superset tipado de JavaScript
  - https://www.typescriptlang.org/

- **ts-node** (MIT) - Blake Embrey e colaboradores
  - Motor de execução TypeScript para Node.js
  - https://typestrong.org/ts-node/

- **nodemon** (MIT) - Remy Sharp
  - Monitor de alterações que reinicia automaticamente a aplicação
  - https://nodemon.io/

- **ESLint** (MIT) - Nicholas C. Zakas e colaboradores
  - Ferramenta de linting para JavaScript/TypeScript
  - https://eslint.org/
